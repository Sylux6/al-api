# al-api
Api that returns azur lane data

api url: https://api.kurozeropb.info/v1/azurlane

| Routes | Param  |
|:------:|:------:|
| /ship  | name   |
| /build | time   |

## Examples
#### /ship
- https://api.kurozeropb.info/v1/azurlane/ship?name=Prinz_Eugen

#### /build
- https://api.kurozeropb.info/v1/azurlane/build?time=00:24:00
